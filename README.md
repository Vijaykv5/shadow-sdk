# Shadow SDK

Shadow SDK is a Solana infrastructure monorepo for private and MEV-resistant execution, perps infrastructure, MagicBlock integration, Jito integration, SDK crates, CLI tooling, relayers, and keeper services.

This repository is intentionally organized as an infrastructure ecosystem, not as a single dApp.
